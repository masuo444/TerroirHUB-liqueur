# Terroir HUB LIQUEUR — 制作ルール

## 1. 情報の正確性（最重要）

### 嘘をつかない原則
- **メーカーが公式に出している情報だけを使う**
- 情報がない項目は「表示しない」。推測で埋めない
- AIが勝手に文章を生成・補完しない

### 引用ルール
- メーカーサイト・公式SNSからの文章引用には**必ず出典を明記**する
  ```html
  <div style="font-size:10px;color:var(--text-muted);margin-top:10px;">
    出典：<a href="https://example.com" target="_blank" rel="noopener">メーカー名 公式サイト</a>
  </div>
  ```
- 杜氏・製造責任者の個人名での発言は、**その人物が実際にその発言をしたと確認できる場合のみ**帰属する
- 確認できない場合はメーカーの一般的な説明として記述する

### データソースの優先順位
1. メーカー公式サイト（最優先）
2. ラベル法定表記（アルコール度数・原材料等）
3. 公式品評会結果（全国梅酒品評会等）
4. 国税庁オープンデータ（メーカー名・住所・創業年）
5. メーカーのSNS公式アカウント

### 使ってはいけないソース
- 個人ブログ・口コミサイトの主観的評価
- AIによる推定・生成テキスト（ファクトチェック不可のもの）
- 出典不明の二次情報

### リキュール固有の情報正確性ルール
- **アルコール度数は公式ラベル通りに記載** — 推測しない
- **ベーススピリッツは公式発表情報のみ** — 味の推測から特定しない
- **果実産地は公式に記載がある場合のみ** — 「おそらく○○産」と推測しない
- **受賞歴は公式発表で確認できるもののみ**

## 2. 銘柄カードの情報設計

### 必須項目（ラベル・公式サイトから取得可能）
- 銘柄名
- 種類（梅酒・ゆず酒・桃酒等）
- アルコール度数
- 主要原料
- 説明文（公式サイトの記述ベース）

### 条件付き表示項目（データがある場合のみ表示）
- **ベーススピリッツ**：ホワイトリカー / 日本酒 / 焼酎 / ブランデー
  - 公式に記載がある場合のみ
- **果実産地**：和歌山県産南高梅 / 四国産ゆず etc.
  - 公式に記載がある場合のみ
- **受賞歴**：年・品評会名・賞名
  - 公式発表で確認できるもののみ
  - 受賞歴がない銘柄にはセクション自体を表示しない

### 画像
- 正方形（aspect-ratio: 1/1）のプレースホルダーを標準装備
- メーカーから使用許可を得た写真 → `<img class="brand-img">` に差替
- 未許可のメーカー → プレースホルダー表示（`brand-img-placeholder`）
- Unsplash等の汎用画像は**メーカーや銘柄の固有ページには使わない**（誤解を招く）
- ポータルトップの一般的なセクション（地域紹介等）ではUnsplash使用可

- **ディスプレイ**：Zen Old Mincho 700（メーカー名・ヒーロー見出し）

- **UI・数値**：DM Sans
- **最小フォントサイズ**：14px（全世代対応）

### レイアウト
- 銘柄カード：3列グリッド、1px gap、ボーダー区切り
- カードホバー：上部にローズピンクグラデーションライン出現
- border-radius：6〜12px
- ボックスシャドウは控えめに（ボーダー主体）

## 5. SEO・AIO戦略

### 構造化データ（必須）
- 全メーカー：LocalBusiness + Product schema
- 教科書ページ：FAQPage schema
- 飲み方ガイド：HowTo schema
- パンくずリスト：BreadcrumbList schema
- サイト検索：SearchAction schema

### AIO（AI Overview）対策
- 教科書ページは「問い→答え」形式で構成
- 1段落200文字以内で結論を先に述べる
- 英語版を必ず用意（英語圏のAI Overviewに引用されるため）
